# SOVA Dataset

SOVA Dataset is free public STT/ASR dataset.

Key facts:
- Russian and English languages
- ~ 28 853 hours
- ~ 2,9 TB in `.wav` format

## Audio characteristics
* Bit rate mode: constant
* Bit rate: 256 kbps
* Channel(s): 1 channel
* Sample rate: 16.0 kHz
* Bit depth: 16 bit
